# Smart Shopping Environment

This environment simulates a real-world shopping assistant.

## Features
- Smart decision making
- Reward shaping
- Multiple actions (buy, skip, compare, wishlist)

## Why this is powerful

- Real-world shopping decision simulation
- Multi-factor reward system (price, rating, discount)
- Adaptive agent behavior
- Supports RL training

## Tasks
- Easy → filter high-quality products
- Medium → budget optimization
- Hard → multi-objective decision making

python inference.py
